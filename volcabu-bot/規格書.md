## 二、功能清單

### ✅ 已實作

#### 【核心】單字查詢
- 用戶傳送任意英文單字（或短語）
- 透過 Hermes API Server → DeepSeek V4 查詢劍橋字典風格解釋
- 回傳格式：
  - 詞性 (part of speech)
  - 英式音標 (UK phonetic)
  - 美式音標 (US phonetic)
  - 英文定義
  - 繁體中文解釋
  - 英文例句 + 中文翻譯

#### 【快取】本地資料庫
- 查過的單字自動存入 SQLite
- 下次查詢同一單字直接回傳（無需重複呼叫 LLM）
- 支援：`/list` 最近查詢、`/stats` 統計

#### 【發音】TTS 按鈕
- 查詢結果附帶「🔊 英式 UK / 🔊 美式 US」按鈕
- 點擊後生成 Google TTS 音檔並傳送

#### 【匯出】CSV / Anki
- `/export` — 匯出完整資料庫為 CSV
- `/export anki` — 匯出 Anki 相容格式 (tab-separated)
- 欄位：word, part_of_speech, definition_en, definition_zh, example_en, example_zh

#### 【安全】用戶管控
- `ALLOW_ALL=true`：允許任何人使用
- `ALLOW_ALL=false` + `ALLOWED_USERS`：僅限特定 Telegram 用戶 ID

## 四、資料庫 Schema

```sql
-- 單字表
CREATE TABLE words (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    word TEXT UNIQUE NOT NULL,        -- 小寫單字
    created_at INTEGER NOT NULL       -- Unix timestamp
);

-- 解釋表（一對多：一個單字可有多個詞性/解釋）
CREATE TABLE definitions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    word_id INTEGER NOT NULL,
    part_of_speech TEXT,              -- noun / verb / adj / adv / etc.
    uk_phonetic TEXT,                 -- 英式音標
    us_phonetic TEXT,                 -- 美式音標
    definition_en TEXT,               -- 英文定義
    definition_zh TEXT,               -- 繁體中文解釋
    example_en TEXT,                  -- 英文例句
    example_zh TEXT,                  -- 中文例句翻譯
    FOREIGN KEY(word_id) REFERENCES words(id) ON DELETE CASCADE
);
```

## 五、API 規格

### 字典查詢（Hermes API Server）

**端點：** `POST /v1/chat/completions`  
**模型：** `hermes-agent`  
**System Prompt：** 要求回傳 JSON 格式的劍橋字典風格解釋  

**請求範例：**
```json
{
  "model": "hermes-agent",
  "messages": [
    {"role": "system", "content": "You are a professional English-Chinese dictionary..."},
    {"role": "user", "content": "apple"}
  ],
  "temperature": 0.2
}
```

**回應範例：**
```json
{
  "word": "apple",
  "results": [
    {
      "part_of_speech": "noun",
      "uk_phonetic": "/ˈæp.əl/",
      "us_phonetic": "/ˈæp.əl/",
      "definition_en": "a round fruit with red, green, or yellow skin...",
      "definition_zh": "蘋果，一種圓形水果...",
      "example_en": "She ate a crisp red apple for lunch.",
      "example_zh": "她午餐吃了一顆清脆的紅蘋果。"
    }
  ]
}
```

## 七、開發備註

- LLM 回傳可能包含 markdown code fence，`dictionary.py` 會自動 strip
- 音檔以 `.ogg` 格式快取，重複查詢同一單字不會重新生成
- 資料庫使用 WAL mode，支援並發讀取
- run.sh 會同時 source Hermes 的 `.env` 和專案的 `.env`，後者優先
